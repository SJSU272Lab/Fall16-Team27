1) As a coach, I would like to login to the application using my Google/Facebook account.

2) As a player, I would like to login to the application using my Google/Facebook account.

3) As a coach, I would like to see a dashboard which displays overview of all my players.

4) As a coach, When I click on any player, I should see all the details/charts and Insights regarding the workout of that particular player.

5) As a coach, I would like to see Graphical representation of different players' workout comparisons. 

6) As a player, I would like to see a dashboard which displays all overview of my periodic(daily/weekly) workouts.

7) As a player, I would like to see my teammates workout statistics. 

8) As a coach, Apart from the Web application I would like to have a mobile app from which I can easily access everything that I can using Web Application.

9) As a Player, Apart from the Web application I would like to have a mobile app from which I can easily access everything that I can using Web Application.
